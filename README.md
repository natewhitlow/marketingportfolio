Data-Driven Visual Portfolio | Nathaniel Whitlow
A high-performance, multi-page professional portfolio built with React and Tailwind CSS. This project showcases the intersection of cinematic production, technical post-processing, and strategic digital marketing.

🚀 Vision
This portfolio is designed to demonstrate more than just "content creation." It is a technical case study in organic growth architecture, specifically highlighting the transition of traditional communities into video-first digital ecosystems.

🛠️ Tech Stack
Framework: React.js

Styling: Tailwind CSS (Modern Dark-Mode UI)

Icons: Lucide-React

Animation: Tailwind Animate (Fade/Slide Transitions)

Deployment: Optimized for Vercel/Netlify

📽️ Key Features
1. Multi-Page Navigation System
Transitioned from a single-page scroller to a dedicated state-based navigation system. This allows recruiters to focus on specific domains: Case Studies, Growth Strategy, and Visual Assets.

2. Color Accuracy & Grading Tool
A custom-built comparison component in the Case Studies section.

Uses synchronized CSS transforms (scale, object-position) to provide a 1:1 comparison of raw vs. graded footage.

Designed to showcase technical proficiency in DaVinci Resolve and color science.

3. High-Retention Content Gallery
Short-Form Engine: Autoplaying, looped vertical frames with user-controlled audio to showcase TikTok/Reels expertise.

Conversion-Centric Thumbnails: Dedicated frames for A/B tested graphic designs aimed at increasing CTR.

Landscape Assets: Clean, 16:9 cinematic production showcase.

4. Strategic Infrastructure Page
A dedicated section detailing the "behind-the-scenes" of growth:

Lead generation funnel visualizations.
